## تامی چیست؟

تامی یک ابزار خودکار راه‌اندازی تونل است که سرور ایرانی شما را به سرور خارجی با سرعت بالا، پایداری و پنهان‌سازی کامل IP متصل می‌کند. از ۷ پروتکل تونل و ۴ روش فوروارد پورت پشتیبانی می‌کند — همه از طریق اسکریپت‌های تعاملی بدون نیاز به ویرایش دستی.

## پروتکل‌ها

| # | پروتکل | انتقال | پنهان‌سازی | سرعت | مناسب برای |
|---|--------|--------|------------|------|------------|
| 1 | **VLESS+Reality** | TCP | ★★★★★ | ★★★★ | بهترین پنهان‌سازی، غیرقابل شناسایی |
| 2 | **Hysteria2** | UDP/QUIC | ★★★★ | ★★★★★ | بالاترین سرعت وقتی TCP محدود شده |
| 3 | **Shadowsocks-2022** | TCP+UDP | ★★★ | ★★★★ | آزمایش‌شده و قابل اطمینان |
| 4 | **TUIC v5** | UDP/QUIC | ★★★ | ★★★★ | تأخیر کم QUIC |
| 5 | **WireGuard** | UDP | ★★ | ★★★★★ | VPN سطح کرنل، سریع‌ترین |
| 6 | **Brook** | TCP+UDP | ★★★ | ★★★ | سبک و ساده |
| 7 | **SSH Tunnel** | TCP | ★★ | ★★ | بدون نیاز به نرم‌افزار اضافی |

## فوروارد پورت

تامی فوروارد پورت را **داخل هر پروتکل** ادغام کرده است. بعد از راه‌اندازی هر تونل، از شما سؤال می‌شود که آیا می‌خواهید پورت‌های خاصی را فوروارد کنید.

| روش | پروتکل‌ها | نحوه کار |
|------|-----------|----------|
| **Xray dokodemo-door** | VLESS+Reality | افزودن اینباندهای شفاف پورت به کانفیگ Xray |
| **SSH -L / -R** | SSH Tunnel | فوروارد محلی و ریموت از طریق autossh |
| **iptables DNAT** | WireGuard | فوروارد NAT سطح کرنل (TCP+UDP) |
| **socat از طریق SOCKS5** | Hysteria2, SS, TUIC, Brook | رله TCP از طریق پروکسی SOCKS5 |

### حالت‌های فوروارد پورت
- **محلی (-L)**: فوروارد یک پورت روی سرور ایران → هاست ریموت از طریق تونل
- **ریموت (-R)**: فوروارد یک پورت روی سرور خارج → بازگشت به سرور ایران
- **چند قانون**: فوروارد چندین پورت به‌صورت همزمان

## شروع سریع

### مرحله ۱: سرور خارج

```bash
# آپلود و اجرا روی سرور خارجی
chmod +x tommy-server-setup.sh
sudo ./tommy-server-setup.sh
```

پروتکل‌ها را انتخاب کنید (پیشنهاد: `1 2 3`). اسکریپت لینک‌های اتصال را چاپ کرده و اطلاعات را در `/root/*-client-info.txt` ذخیره می‌کند.

### مرحله ۲: سرور ایران

```bash
# آپلود و اجرا روی سرور ایرانی
chmod +x tommy-client-iran.sh
sudo ./tommy-client-iran.sh
```

پروتکل مشابه را انتخاب کنید، اطلاعات اتصال از مرحله ۱ را وارد کنید. اسکریپت:
۱. کلاینت تونل را راه‌اندازی می‌کند
۲. درباره فوروارد پورت سؤال می‌کند → در صورت نیاز پیکربندی می‌کند
۳. درباره پروکسی سراسری سؤال می‌کند → در صورت نیاز پیکربندی می‌کند
۴. اتصال را به‌صورت خودکار تست می‌کند

### مرحله ۳: تأیید

```bash
# بررسی IP جدید (باید IP سرور خارجی را نشان دهد)
curl -x socks5h://127.0.0.1:10808 https://api.ipify.org

# یا با پروکسی HTTP
curl -x http://127.0.0.1:10809 https://api.ipify.org

# مقایسه با IP مستقیم
curl https://api.ipify.org
```

## مثال‌های فوروارد پورت

### دسترسی به دیتابیس خارجی از ایران
```
قانون فوروارد: 3306:db.example.com:3306
→ حالا از localhost:3306 روی سرور ایران متصل شوید
```

### نمایش سرویس وب محلی از طریق IP خارجی
```
فوروارد ریموت: foreign_ip:8080 → localhost:3000
→ دسترسی به http://foreign_ip:8080 به سرویس محلی شما می‌رسد
```

### چند پورت همزمان
```
قانون ۱: 3306:db.example.com:3306
قانون ۲: 6379:redis.example.com:6379
قانون ۳: 8080:api.service.com:443
```

## مکان سرور خارج

سرور خارج نزدیک به ایران انتخاب کنید برای کمترین تأخیر:

| مکان | تأخیر | توضیحات |
|------|-------|---------|
| ترکیه (استانبول) | ۲۰-۴۰ میلی‌ثانیه | نزدیک‌ترین، برخی محدودیت‌های ISP |
| آلمان (فرانکفورت) | ۶۰-۹۰ میلی‌ثانیه | اتصال خوب |
| هلند (آمستردام) | ۷۰-۱۰۰ میلی‌ثانیه | محبوب برای حریم خصوصی |
| فرانسه (پاریس) | ۶۵-۹۵ میلی‌ثانیه | مسیریابی خوب خاورمیانه |

## مدیریت سرویس‌ها

همه سرویس‌ها از پیشوند `tommy-` استفاده می‌کنند:

```bash
# بررسی وضعیت
systemctl status tommy-xray
systemctl status tommy-hysteria
systemctl status tommy-shadowsocks
systemctl status tommy-tuic
systemctl status tommy-brook
systemctl status tommy-ssh

# راه‌اندازی مجدد
systemctl restart tommy-xray

# مشاهده لاگ‌ها
journalctl -u tommy-xray -f

# سرویس‌های فوروارد پورت
systemctl status tommy-pf-ssh-3306
systemctl status tommy-pf-socat-8080
```

## نکات امنیتی

- VLESS+Reality بیشترین پنهان‌سازی را دارد — ترافیک شبیه HTTPS عادی است
- Hysteria2 وقتی پروب می‌شود خود را سرور وب جا می‌زند
- تونل‌های SSH قابل شناسایی هستند اما نیاز به نرم‌افزار اضافی ندارند
- WireGuard سریع‌ترین اما قابل شناسایی‌ترین است
- همیشه از احراز هویت کلید SSH استفاده کنید
- اطلاعات اتصال را به‌طور دوره‌ای تغییر دهید

## پیش‌نیازها

- **سرور خارج**: اوبونتو ۲۰.۰۴+ / دبیان ۱۱+ / سنت‌اواس ۸+ / راکی لینوکس
- **سرور ایران**: همان نیازمندی‌های سیستم‌عامل
- **دسترسی root** روی هر دو سرور
- **پورت‌های خروجی باز** روی سرور خارج (۴۴۳/TCP، ۸۴۴۳/UDP و غیره)
